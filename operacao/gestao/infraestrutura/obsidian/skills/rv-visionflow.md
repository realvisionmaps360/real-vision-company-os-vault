# rv-visionflow

> Skill manual de segurança do VisionFlow (CRM). Arquivo real: `.claude-skills/rv-visionflow/SKILL.md`

## O que faz

Manual de pré-voo para qualquer sessão que tocar o banco do VisionFlow. Contém:

- Checklist obrigatório antes de mexer no banco (git status primeiro)
- Valores válidos para cada campo (status, tipo de serviço, etc.)
- Pitfalls comuns e como evitar

## Como usar

```
/rv-visionflow
```

Carregar antes de qualquer sessão que envolva banco de dados do CRM.
