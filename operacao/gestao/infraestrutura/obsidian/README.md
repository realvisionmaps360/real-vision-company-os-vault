## Resumo de 30 segundos

- As notas (`.md`) do Company OS são sincronizadas entre **computador** e **celular** usando o **GitHub** como ponte.
- Repositório **privado**: `real-vision-company-os-vault` (conta `realvisionmaps360`).
- Sobe **só texto** (`.md`); imagens, vídeos, PDFs e repos de código ficam fora.
- Tudo automático a cada 5 minutos. Se algo parar, ver [[GUIA-Sincronizacao]] → "Se algo parar".
