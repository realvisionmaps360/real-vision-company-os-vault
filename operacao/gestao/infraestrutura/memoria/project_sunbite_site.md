**SUNBITE** é a marca premium suíça de sobremesas da Romana (parceira pessoal do Felipe). O site é o próximo projeto a ser desenvolvido (ou refeito com o Master Brief completo).

**Domínio:** sunbite.ch  
**Briefing completo:** `Desktop\Real Vision\operacao\projetos\BRIEFING-SUNBITE.md`

**Posicionamento central:**
SUNBITE não vende morangos — vende uma *experiência premium de verão suíça*. Marca de lifestyle, não food truck. Deve passar no "Final Brand Test": *"Does this feel like a premium Swiss summer dessert experience?"*

**Produto:** Strawberry Chocolate Cups (chocolate suíço premium + morangos frescos, preparado na hora). Um produto feito excepcionalmente bem.

**Slogan:** *Made for Sunny Days*

**Cores:** vermelho morango `#8c090d` / creme `#F5E9DA` / baunilha `#F2D16B`

**Fontes de receita:** walk-in + eventos (casamentos, corporativo, festivais, aniversários) — o site precisa de CTA de reserva de eventos.

**4 objetivos do site:**
1. Fazer querer experimentar
2. Gerar confiança na qualidade
3. Fazer seguir no Instagram
4. Fazer reservar para eventos

**Seções planejadas:** Hero · Product Story · Ingredient Story · About Romana · Where To Find Us · Upcoming Events · Book SUNBITE · Gallery · Instagram · FAQ · Contact

**Stack:** React + Vite + TypeScript + Tailwind + framer-motion + Vercel

**Prazo:** antes do fim de junho/2026 (Romana chega com a food bike)

**Status atual (09/06/2026):** primeira versão do site construída (inglês, 5 seções básicas). Master Brief completo recebido — Felipe vai enviar prompt para refazer com posicionamento correto.

**Pendente:** Instagram handle oficial + fotos da food bike (confirmar com Romana)

**Why:** Primeiro projeto de cliente 100% com Claude Code (sem Lovable). Romana é parceira pessoal do Felipe.

**How to apply:** quando o assunto for "site da Sunbite" ou "site da Romana", ir direto ao briefing. O posicionamento PREMIUM é inegociável — nunca sugerir design genérico, carnival-like ou barato.
