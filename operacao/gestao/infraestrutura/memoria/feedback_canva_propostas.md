Quando Felipe pedir para criar **proposta comercial** para qualquer cliente, usar o **MCP do Canva** para gerar o design visualmente.

**Why:** Felipe quer propostas com design profissional, não só texto. O Canva MCP está conectado e permite criar e exportar designs diretamente do chat.

**How to apply:** Em sessões de proposta (skill `proposta-comercial`), acionar o Canva MCP para criar a capa e slides visuais. Verificar brand kits disponíveis antes de gerar. Lembrar de usar junto com a skill `proposta-comercial` e dados do cliente.
