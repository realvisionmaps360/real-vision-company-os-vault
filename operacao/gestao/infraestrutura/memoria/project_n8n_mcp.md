# n8n-MCP — Instalar no futuro

## O que é
MCP server que conecta Claude ao n8n (plataforma de automação de fluxos, open source, tipo Zapier/Make).
Repo: https://github.com/czlonkowski/n8n-mcp

## O que oferece
- Acesso a 1.851 blocos de automação (nodes)
- 2.352 templates de workflow prontos
- 87% de cobertura da documentação oficial do n8n
- Validação de configurações de nodes

## Quando instalar
Quando a Real Vision adotar n8n para automações internas — ex: formulário do site → notifica WhatsApp → salva no VisionFlow, ou qualquer fluxo de automação de processo interno.

## Como instalar (quando chegar a hora)
1. Verificar se n8n está rodando localmente ou via cloud
2. Instalar o MCP server: seguir instruções em https://github.com/czlonkowski/n8n-mcp
3. Adicionar ao settings.json em mcpServers
