Expansão das redes sociais da Real Vision (Instagram + TikTok) está planejada como próxima fase — ainda não iniciada (mai/2026). Felipe sempre deixou essas plataformas para trás; com a organização via Claude Code em andamento, o momento está se aproximando.

**Why:** Real Vision focou em presença local/Google até agora. Redes sociais são canal de aquisição inexplorado que vai alimentar a fase de escala (conteúdo, curso, rede de fotógrafos).

---

## Ferramenta central: Windsor AI

**Windsor AI** é uma plataforma de analytics que funciona como conector MCP dentro do Claude.ai. O mesmo conector serve para Instagram e TikTok. É gratuito.

O que entrega além das métricas nativas das plataformas:
- Top vídeos/Reels com melhor performance + por que viralizaram
- Lista de conteúdos para **parar**, **repetir** e **escalar**
- Diagnóstico de gargalos que limitam o crescimento
- Plano de ação personalizado para os próximos 30 dias
- Ideias de conteúdo baseadas nas métricas do próprio perfil

---

## Passo a passo — Instagram

1. Acessar **claude.ai** → ir em **Personalizar conectores** → clicar no **+** para adicionar novo conector
2. Buscar por **Windsor AI** → instalar → aceitar permissões
3. Dentro do Windsor AI, selecionar **Instagram Insights** → clicar no botão azul para conectar
4. Prompt de uso: _"Usando o conector Windsor AI, me dê as métricas do meu perfil no Instagram e com base nelas me fale exatamente o que eu preciso melhorar"_

---

## Passo a passo — TikTok

1. Ir ao **Google** → pesquisar **Windsor AI** → entrar no primeiro link (windsorapp.co ou similar)
2. Fazer login com conta Google
3. Na barra de busca da plataforma, digitar **TikTok** → selecionar **TikTok Organic** → clicar no botão azul → clicar em **Continuar**
4. Abrir **Claude.ai** → **Personalizar** → adicionar conector **Windsor AI** (o mesmo já integrado)
5. Prompt de uso: _"Usando o Windsor AI, analise as métricas do meu TikTok e fale como eu posso melhorar com foco em [vendas / ganho de seguidores / etc]"_

---

## Observação importante

O Windsor AI **não é um MCP técnico instalado no Claude Code CLI** — é um conector nativo do **Claude.ai (web)**. Portanto, quando for usar, acessar via claude.ai no browser, não pelo terminal do Claude Code.
