O PC do Felipe tem uma **GTX 1060 com 3 GB de VRAM** (confirmado 04/07/2026). Modelos de IA de ~8B parâmetros não rodam localmente (precisam de 6–16 GB de VRAM). Antes de sugerir rodar qualquer modelo local (TTS, imagem, LLM), verificar se cabe em 3 GB; senão, propor GPU de nuvem por hora (RunPod/Modal/Colab) ou API paga. Contexto de origem: avaliação do MOSS-TTSD para [[project-jogo-da-terra]].
