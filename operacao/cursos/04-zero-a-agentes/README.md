# Do Zero a Agentes: Curso Prático de IA para Pequenas Empresas

**Status:** 🔨 Em Estruturação (01/07/2026)

---

## Visão Geral

Curso de capacitação profissional para colaboradores de pequenas empresas e PMEs. Objetivo: transformar funcionários não-técnicos em utilizadores práticos de IA e Claude Code, capazes de automatizar fluxos de trabalho e gerenciar nuvem.

**Público-alvo:** Colaboradores de empresas (clientes Real Vision) que não têm conhecimento prévio em IA, Claude Code ou cloud computing.
