# O que é este arquivo — Tour Virtual 360°

Este arquivo (`05-tour-virtual-360.zip`) é a pasta completa de exportação do tour virtual 360° produzido para a empresa, feito com câmera 360° e software especializado (Pano2VR).

## O que tem dentro

O tour interativo completo: as imagens 360°, os pontos de navegação (hotspots) e os arquivos de visualização (HTML/JS) que fazem o tour funcionar num navegador.

## O que fazer com ele

Extraia o zip e hospede a pasta em qualquer serviço de hospedagem estática, ou peça a um desenvolvedor para embutir esse tour dentro do novo site (por exemplo, via iframe).

## Observação

É um site estático — não precisa de banco de dados nem de conta paga específica para funcionar, só de um lugar para hospedar os arquivos.
