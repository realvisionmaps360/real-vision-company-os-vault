# O que é este arquivo — Chatbot

Este arquivo (`04-chatbot-fluxo-automacao.rar`) é o export do fluxo de automação (chatbot) que atendia no site, feito na plataforma n8n.

## O que fazer com ele

Para usar de novo, alguém técnico precisa ter (ou criar) uma conta no n8n — existe uma versão gratuita para auto-hospedar e uma versão paga em nuvem — e importar esse arquivo como um novo "workflow" (fluxo de automação).

## Observação

As integrações de mensagens (WhatsApp, número de telefone, etc., se houver) precisarão ser reconectadas do zero na conta nova, porque estavam ligadas a uma conta que não é repassada.
