# Glossário — Real Vision Academy

> Termos do projeto, para humanos e IA falarem a mesma língua.

| Termo | Significado |
|---|---|
| **Academy** | Real Vision Academy — plataforma de ensino/membros, rota `/academy` do site |
| **Curso avulso** | Compra única de um curso (sem assinatura) |
| **Matrícula (`enrollment`)** | Vínculo que dá a um usuário acesso a um curso |
| **Área de membros** | Área logada onde o aluno vê e consome os cursos adquiridos |
| **Material complementar** | Anexo de uma aula (prompt `.md`, PDF, link, download) |
| **Prompt `.md`** | Material em Markdown com prompts prontos — produto destaque da Academy |
| **AuthContext** | Provider global de sessão do usuário (a criar; hoje inexistente) |
| **Usuário final / público** | Visitante do site/blog e aluno da Academy (banco Supabase novo) |
| **CRM interno** | VisionFlow — sistema interno da RV, no Supabase `ghwjetvazmdlaqidgxqi` (separado) |
| **Gaveta B** | Documentação viva no vault Obsidian (vs. "Gaveta A" = junto do código) |
| **Master Visionair** | Skill arquiteta-documentadora que conduz este projeto |
