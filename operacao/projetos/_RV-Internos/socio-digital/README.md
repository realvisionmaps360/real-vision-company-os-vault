## O que é

Sócio Digital é o **6º pilar de serviço** da Real Vision. Implementamos remotamente no computador do cliente: Claude Code + plano Pro + estrutura organizacional (Company OS adaptado) + treinamento ao vivo + acompanhamento mensal com insight estratégico.

Cliente passa a delegar tarefas operacionais à IA, mantendo controle estratégico.
