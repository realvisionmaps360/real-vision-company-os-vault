# Pipeline de Produção

```
Ideia (banco-de-ideias) 
  → Priorização (backlog-priorizado)
  → Pauta + roteiro (youtube-pesquisar-pauta + youtube-criar-video)
  → Gravação (Felipe, fora do Claude Code)
  → Pacote de publicação (youtube-revisar-pacote)
  → Aprovação Nível D (Felipe)
  → Publicação (youtube-publicar, quando MCP existir)
  → Acompanhamento (youtube-analise-semanal)
```

Cada seta corresponde a um arquivo novo ou atualizado na pasta do vídeo (`04_videos/AAAA-MM-DD_slug/`), nunca só conversa.
