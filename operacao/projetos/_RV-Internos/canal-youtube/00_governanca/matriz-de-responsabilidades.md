# Matriz de Responsabilidades

Operação é conduzida por Felipe Garcia como único operador no momento (ver `contexto/TIME.md`). Esta matriz existe para quando a equipe crescer.

| Responsabilidade | Quem decide hoje | Quem executa hoje |
|---|---|---|
| Estratégia/pilares do canal | Felipe | Felipe + Claude (proposta) |
| Aprovação de roteiro/pauta | Felipe | Claude prepara, Felipe aprova |
| Gravação/edição | Felipe | Felipe |
| Publicação (Nível D) | Felipe (aprovação explícita) | Claude executa só após aprovação |
| Resposta a comentário (Nível C) | Felipe | Claude prepara, Felipe aprova cada resposta |
| Análise semanal/mensal | Felipe consome | Claude prepara |
| Exclusão de qualquer recurso (Nível E) | Felipe (dupla aprovação) | Claude executa só após as duas aprovações |

Qualquer mudança nesta matriz (ex.: nova contratação assumindo parte disso) é decisão registrada em `07_decisoes/registro-de-decisoes.md`.
